## Typora+PicGo+GitHub

### Github配置

1.  新建或使用已有仓库
2.  仓库属性必须为**公共**（public）
3.  设置`Developer settings`（https://github.com/settings/tokens）
4.  保存token，这个 token 只显示一次！！！
